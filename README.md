# Consumer Insights Hub - Bayer OTC US

Hub interactivo de insights para Bayer OTC US, desarrollado por **Maggiore**.

## Estructura

```
hub-bayer/
  index.html   - Estructura HTML y contenido
  style.css    - Estilos visuales
  script.js    - Datos, logica y funcionalidades
  README.md    - Este archivo
```

## Credenciales de acceso

- **Acceso:** protegido por clave
- **Clave:** compartida internamente por Maggiore

## Actualizacion automatica

El Hub se conecta a una Google Sheet publicada y actualiza los datos (IEP y SL) cada vez que se abre. No es necesario re-subir archivos para cambios de datos.

## Deploy

Subir los 3 archivos (`index.html`, `style.css`, `script.js`) a la carpeta `public_html/hub-bayer/` en Hostinger, o usar GitHub Pages.
